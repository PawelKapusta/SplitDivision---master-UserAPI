# SplitDivision---master-UserAPI

### Running the development server.

### To install

```bash
    npm install
```

### And run:

```bash
    npm run start
```

### To run project

#### Build docker image

```bash
    docker build -t master-user-api .
```

#### Run a container on port 5002

```bash
    docker run -p 5001:5001 master-user-api
```
